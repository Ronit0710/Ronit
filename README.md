Developed an interactive Financial Dashboard in Microsoft Excel to analyze revenue, expenses, profit, and overall business performance. Leveraged Pivot Tables, Pivot Charts, Slicers, and data visualization techniques to create an intuitive dashboard for financial analysis and decision-making.

Key Features:
• Revenue, Expense & Profit Analysis
• Category-wise Financial Breakdown
• Monthly Profit Trend Analysis
• Interactive Filters using Slicers
• KPI-based Dashboard Visualization
